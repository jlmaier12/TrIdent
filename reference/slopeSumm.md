# Summarize slopes for sloping classifications

Add slope information for sloping classifications to summary table

## Usage

``` r
slopeSumm(classifSumm, slopingClassifList, windowSize)
```

## Arguments

- classifSumm:

  Classification summary table

- slopingClassifList:

  A list containing pattern match information associated with all
  contigs classified as sloping.

- windowSize:

  The window size used to re-average read coverage pileups

## Value

dataframe
